# W3Schools Homepage Clone

A responsive clone of the W3Schools homepage built using plain HTML, CSS, and JavaScript.

## About The Project

This project is a front-end recreation of the W3Schools website homepage. It demonstrates modern web development techniques using core web technologies without any frameworks or libraries.

## Features

- **Responsive Design:** Fully responsive layout that works on desktop, tablet, and mobile devices
- **Pure Technologies:** Built with vanilla HTML, CSS, and JavaScript
- **Modern UI:** Replicates the clean and user-friendly interface of W3Schools
- **Interactive Elements:** Includes navigation menus, hover effects, and other interactive components

## Live Demo

Check out the live demo here:

[https://ajay117.github.io/w3schools-clone/](https://ajay117.github.io/w3schools-clone/)

## Technologies Used

- **HTML5:** For structure and semantic markup
- **CSS3:** For styling, layouts, and responsive design
- **JavaScript:** For interactive functionality

## Responsive Design

The website adapts to different screen sizes:

- **Desktop:** Full navigation with sidebar and main content area
- **Tablet:** Optimized layout for medium screens
- **Mobile:** Collapsible navigation and stacked content for small screens

## Purpose

This project was created to practice and demonstrate:

- Responsive web design principles
- CSS Grid and Flexbox layouts
- JavaScript DOM manipulation
- Cross-browser compatibility
- Clean code organization

## Installation

To run this project locally:

1.  Clone the repository:

    git clone https://github.com/ajay117/w3schools-clone.git

2.  Navigate to the project directory:

    cd w3schools-clone

3.  Open index.html in your web browser
